# Superstore Sales Data Analysis & Sales Forecasting

## Objective
The objective of this project is to perform a detailed analysis of **Superstore sales data** using **Power BI** and generate actionable insights into sales, profit, customer behavior, and returns. Additionally, the **Power BI Forecast tool** is applied to predict future sales trends, enabling better business planning and resource allocation.

## Project Description
This repository contains:
- **Superstore dataset** (sales, profit, returns, shipping, payment modes, regions, states).
- **Power BI dashboards** for interactive analysis.
- **Forecasting models** built using Power BI’s forecasting capability.

The analysis explores:
- Regional and state‑level performance.
- Profitability by payment and shipping modes.
- Customer returns and their impact on revenue.
- Forecasted sales trends for strategic decision‑making.

## Key Insights
- **East & Central regions** lead in sales, while **West & South** show potential for improvement.
- **Card payments** generate the highest profits, followed by Online; **COD** lags behind.
- **Standard Class shipping** is the most profitable, while **Same Day shipping** incurs higher costs.
- **California, New York, and Texas** dominate in both sales and profit.
- **Returns are minimal (~5%)**, but still represent significant lost revenue worth monitoring.

## Forecasting
Using the **Power BI Forecast tool**, future sales are projected based on historical data. This helps:
- Anticipate demand fluctuations.
- Plan inventory and logistics.
- Support growth strategies with data‑driven decisions.

## Dashboards
The project includes interactive dashboards such as:
- Regional & Segment Sales
- Profit by Payment Mode
- Sales & Profit by State
- Profit by Ship Mode
- Returns Impact Analysis
- Geographic Sales & Profit Map
